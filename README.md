# Installation Guide


```js
git clone https://github.com/Yourstruggle11/CaptureTheScreen.git
```

```js
cd ./CaptureTheScreen
```


# Next

- Open Chrome Browser
- Go to [chrome://extensions/](chrome://extensions/)
- Toggle the top right button to enable Developer Mode.. (See the Screenshot below)
